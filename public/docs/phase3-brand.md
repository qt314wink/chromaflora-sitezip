## Brand Manifesto

**ChromaFlora exists because beauty is not decoration.**

Most software separates the tools from the philosophy — you're given a canvas but no cosmology. You're given a sequencer but no theory of rhythm. You're given an AI generator but no grammar for interpreting what it gives back.

ChromaFlora refuses this. Every feature here emerges from a question:

*What if the design system itself was a thinking system?*

We built color tokens that map to emotional frequencies. We built a rhythm generator that uses the same algorithm that places fuel rods in a nuclear reactor — because mathematics doesn't care about genre. We built a discovery engine that actively fights against your own preferences — because curiosity degrades without exercise.

This is not a tool. It is a living argument.

The argument is this: **form is content. System is meaning. Aesthetic is infrastructure.**

If you're making something with ChromaFlora, you are making something with a philosophy embedded in every pixel, every beat, every glow.

Welcome to the void that glows.

### Long Bio (500 words) — For Website About Page
Jennipher Troup didn't set out to build a design system. She set out to answer a question: *what does a design system look like when it's built by someone who sees the same structure in everything?*

The question came from frustration. Every tool she used — Figma for design, Ableton for music, Midjourney for image generation — treated creativity as a series of isolated acts. You design here. You compose there. You generate elsewhere. The connections between them — the way a mandala's 8-fold symmetry maps to the harmonic overtone series, the way the logistic map's chaos appears in both drum patterns and turbulent flows — were invisible to the software.

ChromaFlora was built to make those connections visible.

The name is a compound: Chroma (color, from the Greek χρῶμα — the property of light that makes things visible) + Flora (from the Latin flos — plants, organic systems, growth). The system is additive light (bioluminescent, deep-sea, glowing from within) applied to organic growth (petals, spirals, mycelial networks). Color as life. Light as growth. Design as biology.

The technical execution required building across multiple domains simultaneously: Canvas 2D for generative art, Web Audio API for synthesis, Tone.js for rhythm generation (specifically the Bjorklund algorithm — a neutron physics tool repurposed for drum patterns), Three.js for WebGL particle systems, and the Pollinations.ai free API for image generation.

The philosophical execution required something harder: committing to a small set of constraints. Six colors. Two fonts. One background. No images, only procedures. Glow is the shadow system. Everything that looks complex must be derivable from the simplest possible rule.

This is what Jennipher calls *systems thinking as aesthetic practice*: the discipline of deriving maximum complexity from minimum rules, and proving that constraint is not limitation — it's the condition for creativity.

ChromaFlora now has eight modules:
- **Studio**: Canvas 2D generative painting with physics-based pour and sacred geometry overlays
- **Mandala**: Real-time polar math rendering with simplex noise turbulence
- **Sonic Waves**: Web Audio API visualization with frequency analysis
- **Band**: Tone.js step sequencer with Euclidean, chaos, and groove algorithms
- **AI Studio**: Pollinations.ai image generation + Reverse Image Engineering
- **Discovery**: Anti-feedback-loop inquiry engine across 9 knowledge territories
- **ASMR**: Binaural tone synthesis with visual mandala meditation
- **Embed**: Portable iframe system for sharing living artifacts

Each module is a complete expression of the philosophy. Each one can be used in isolation. Together, they form an argument.

Jennipher is available for: creative technology consulting, design system architecture, generative art commissions, speaking on systems thinking and cross-domain creativity, and white-label ChromaFlora builds for studios and brands.

## Photography + Art Direction for Brand Assets

### Color Palette for Brand Photography
All photography should feel like it was taken in the ocean at depth, or in deep space:
- Dominant: Deep blue-black (#030310 equivalent)
- Accent: Bioluminescent violet/aqua glows
- Subject: Objects that glow from within (jellyfish, fiber optics, night city lights)
- Never: Harsh sunlight, pure white backgrounds, warm yellow tones

### Approved Image Types
✅ Deep-sea bioluminescent organisms
✅ Nebula photography (Hubble + Webb)
✅ Microscopy images (diatoms, mycelium, crystal structures)
✅ Long-exposure night photography
✅ Fiber optic installations
✅ Mandalas and sacred geometry photography
✅ Cymatics (physical wave patterns in sand/fluid)

❌ Studio portrait on white background
❌ Flat-lay product photography
❌ Stock photography with visible models/people
❌ Bright daylight photography
